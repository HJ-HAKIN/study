Q. 함수 호출시 new 를 붙인 것과 안 붙인 것의 차이를 설명하세요.

예시)
var obj = function () {
   console.log(this);
};
obj();
// new obj();

obj();를 실행하면 해당 함수 안의 내용만 실행하지만,
new obj();를 실행하면 해당 함수를 저장하여 실행한다.
